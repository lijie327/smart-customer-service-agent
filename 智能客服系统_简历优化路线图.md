## 四、面试官常见追问 & 标准答法

- **为什么多 Agent 而不是一个大 Prompt？** 职责隔离、可单独评测与迭代、工具权限隔离、便于扩展新业务线。
- **路由怎么做的？** 双引擎：关键词快路径（省 LLM 调用）+ LLM 语义分类，并用对话历史做上下文感知，避免短句误路由。
- **怎么防幻觉？** 规则优先（退款/订单走确定性逻辑）+ RAG 引用约束 + 低置信度转人工。
- **流式怎么实现？** SSE + async generator，后端 `await` 真实 LLM stream，逐 token 推送。
- **性能瓶颈在哪？** 同步 LLM 调用阻塞事件循环（已异步化）；向量检索的维度 / 索引类型选择。
